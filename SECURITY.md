# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

We take the security of `touchstone.parser` seriously. If you believe you have found a security vulnerability, please report it to us by emailing suryaraj@example.com.

Please do not open a public issue for security vulnerabilities.

We will acknowledge your email within 48 hours and provide a timeline for addressing the issue.
